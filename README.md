# Libre-se
Projeto desenvolvido para atuar como um Tradutor de Texto para Língua Brasileira de Sinais (LIBRAS).

Sobre o Projeto
O LIBRE-SE é um projeto desenvolvido para atuar como um tradutor de texto para a Língua Brasileira de Sinais (LIBRAS).

O objetivo principal é oferecer uma interface simples, intuitiva e acessível, permitindo que o usuário digite um texto e visualize a tradução em LIBRAS através da integração com o widget VLibras.
